Título modal

Este es un contenido de marcador de posición para mostrar un modal centrado verticalmente. Hemos agregado una copia adicional aquí para mostrar cómo funciona el centrado vertical del modal cuando se combina con modales desplazables. También utilizamos algunos saltos de línea repetidos para ampliar rápidamente la altura del contenido, activando así el desplazamiento. Cuando el contenido supera la altura máxima predefinida del modal, el contenido se recortará y se podrá desplazar dentro del modal.
